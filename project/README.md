## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Add bootstrap e bootstrap-vue

```sh
npm i bootstrap bootstrap-vue
```

### Add fabric e vue-fabric

```sh
npm i fabric vue-fabric
```

### Add sass

```sh
npm i sass
```
